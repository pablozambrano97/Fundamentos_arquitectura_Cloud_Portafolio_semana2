# Fundamentos_arquitectura_Cloud_Portafolio_semana2
El objetivo del proyecto es diseñar y construir una arquitectura para un
sistema de reservas en la nube que sea escalable, seguro y eficiente. La
aplicación debe ser capaz de adaptarse a diferentes cargas de usuarios y
asegurar un tiempo de respuesta rápido. Además, el sistema debe
garantizar la seguridad de los datos de los usuarios y permitir
modificaciones sin afectar la estructura principal.
